# 体育器材借阅系统

这是一个最小的借阅系统示例，使用 Flask + SQLite。

准备与运行（Windows PowerShell）:

1. 创建并激活虚拟环境：

```powershell
python -m venv .venv; .\.venv\Scripts\Activate.ps1
```

2. 安装依赖：

```powershell
pip install -r requirements.txt
```

3. 初始化数据库：

```powershell
python db.py
```

4. 运行应用：

```powershell
python app.py
```
